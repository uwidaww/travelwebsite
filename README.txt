If you want to open this project, make sure you have
XAMPP installed on your device.

----

Steps to open this project :
1.  Put the project folder to htdocs folder under XAMPP folder
    (usually you will save it on C:/xampp/htdocs)
2.  Start the Apache and MySQL module on your XAMPP application
3.  Open your prefered browser and type localhost/(thisfilename)
    on the searchbar
